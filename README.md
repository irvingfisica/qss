# Ejercicios del libro Quantitative Social Science: An Introduction, de Kosuke Imai

Repo para realizar los ejercicios del libro [Quantitative Social Science: An Introduction](https://press.princeton.edu/books/hardcover/9780691167039/quantitative-social-science). 

La web del libro se encuentra [aquí](http://qss.princeton.press) y el repositorio con los datos para los ejercicios se encuentra [aquí](https://github.com/kosukeimai/qss).

Originalmente el libro propone resolver los ejercicios en R, en este repo se resuelven en Rust con el objetivo de implementar soluciones para realizar análisis de datos.
